## Installation

### Requirements

* Python 3.10 or higher
* pip

### Install from source (development mode)

Clone the repository and install in editable mode:

```bash
git clone git@github.com:danymukesha/AAIE.git
cd AAIE
pip install -e .
```

This installs the `aaie` CLI locally.

### Verify installation

```bash
aaie --help
```

If successful, the CLI command list will be displayed.
